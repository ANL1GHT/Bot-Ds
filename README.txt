Основной код бота в папке cod_bot, главный файл - main.py

установщик python нужен дял его скачивание перед запуском бота
ffmpeg.exe для работы бота
в папке venv содержать все модули для его работы
папка tracks для треков
остальные папки - побочные файлы python


Чтобы включить код для бота нужно его привязать к боту в дс(его можно создать на портале discord developers)
	и подключить токен бота прописав его в файле в файле config.py в папке cod_bot в строку token

Инструкция по запуску
Устанавливаем python 3.10 на компьютер

запускаем IDE для python либо cmd

IDE:Открываем файл main.py и запускаем

cmd:Устанавливаем полный путь к файлу main.py  пример: cd C:\Users\Serg\source\PepsiBot

Запускаем файл командой python main.py

в файле config.py в папке cod_bot указаны токен бота и его префикc,
 если хотим изменить конфигурацию на работу бота с треками не с ютуб а с компьютера, пишем в config.py вместо configuration = 'Bot'
											configuration = 'FileBot'

и треки сохраняем в папке tracks
